# express - template

## 구현목록

- session 을 이용한 로그인 처리
- sequlize
- 메모장 CRUD

## 설치 패키지

```
  npm i express # 익스프레스
  npm i morgan # 로그 기록
  npm i nodemon # 자동 재실행
  npm i cookie-parser # 쿠키 해석
  npm i express-session # 세션
  npm i dotenv # 설정파일
  npm i nunjucks # 템플릿 엔진
```

## 실행방법

### 도커 mysql 실행

### 앱 서버 실행

```
  npm run start
```
